# Unified Analytics & Lead Growth

> **Domain:** Marketing / AdTech

## Overview

The business needed a single, trustworthy analytics backbone to replace scattered spreadsheets, slow reporting, and unclear ownership of data. Key pain points: missing documentation of current databases, inconsistent definitions across teams, manual effort to prepare weekly performance views. Lead-generation performance was hard to diagnose because historic data was not modeled for analysis. Without change, decisions would continue to be delayed, campaigns misallocated, and growth opportunities missed while data debt, rework, and operational risk kept rising.

## Approach

- Audited existing databases and workflows; documented entities, lineage, data owners to establish shared source of truth
- Redesigned data architecture (staging, warehouse, marts) with star schema for marketing, sales, funnel analytics
- Built ELT/ETL pipelines from operational systems; standardized cleaning, enrichment, slowly changing dimensions for history
- Orchestrated analytics pipelines with scheduling, alerts, data quality checks (freshness, completeness, reconciliation to source)
- Backtested and improved lead-scoring approach using previous-year data; added features and set thresholds for action
- Delivered self-serve dashboards for executives and operators; iterated in sprints with UAT and sign-offs
